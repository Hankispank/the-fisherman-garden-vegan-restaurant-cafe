Book or order programmatically

Agents and integrations can submit orders and table reservations to our public endpoint. Responses include a reference id on success.

Endpoint

POST /.netlify/functions/submit-order

OpenAPI 3.1 specification · Full site summary (llms-full.txt)

Example

curl -X POST "https://the-fisherman-garden-vegan-restaurant-cafe.netlify.app/.netlify/functions/submit-order" \
  -H "Content-Type: application/json" \
  -d '{"type":"reservation","customer_name":"Anna","customer_contact":"+84901234567","date":"2026-07-10","time":"19:00","guests":"2","contact_method":"phone","channel":"zalo","botcheck":""}'

Limits

Maximum 30 submissions per IP address per 15 minutes. Request body limit 10 KB. Include an empty botcheck field; do not populate honeypot fields.
